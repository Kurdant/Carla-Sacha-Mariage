## ✍️ PATTERNS TYPOGRAPHIQUES

### Observation des moodboards

**1. Dualité Script + Sans-serif**
- **Noms du couple** : Police **script élégante** (calligraphique, romantique)
  - Exemples observés : "Fernanda & Gustavo", "Михаил & Ольга"
  - Style : Fluide, cursive, féminine sans être excessive
  - Usage : Titres principaux, hero banner, noms uniquement

- **Texte courant** : Police **sans-serif moderne**
  - Exemples observés : Texte du site, labels, descriptions
  - Style : Épuré, lisible, élégant et contemporain
  - Usage : Paragraphes, labels, boutons, infos pratiques

**2. Hiérarchie typographique observée**
- **H1 (Noms couple)** : Script, grande taille, spacing généreux
- **H2 (Sections)** : Sans-serif, uppercase avec letter-spacing, size modérée
- **Body** : Sans-serif, weight 300-400, line-height confortable (1.6)
- **Captions** : Sans-serif, size réduite, weight light

**3. Letter-spacing**
- Uppercase sections : Letter-spacing augmenté (+0.1em à +0.2em)
- Script : Letter-spacing naturel
- Dates/Nombres : Tabular nums, spacing ajusté

## 📐 COMPOSITION & LAYOUT PATTERNS

### 1. Structure Mobile-First Dominante
- **Tous les exemples** montrent des designs **verticaux optimisés mobile**
- Sections empilées (stacking naturel)
- Images pleine largeur ou contenues dans padding généreux
- Touch targets respectés (boutons min 44x44px)

### 2. Espaces Blancs Généreux
- **Ratio observé : 70% espace / 30% contenu**
- Padding sections : Large (60-80px sur mobile, 120-160px sur desktop)
- Marges entre éléments : Système cohérent (multiples de 8px probable)
- Respiration verticale : Très présente entre sections

### 3. Sections Alternées
**Pattern récurrent** :
```
Section 1 : Fond Blanc cassé / Texte Olivewood
Section 2 : Fond Olive/Sage / Texte Parchment
Section 3 : Fond Blanc cassé / Texte Olivewood
Section 4 : Fond Sand / Texte Olivewood
```
Alternance couleur/blanc pour créer rythme et hiérarchie visuelle.

### 4. Grid & Alignement
- Centrage horizontal dominant
- Alignement texte : Centre pour titres, gauche pour paragraphes
- Photos : Pleine largeur ou centrées avec border-radius subtil (8-16px)

### 5. Cards & Containers
- Bordures subtiles (1px Sage/Olive)
- Border-radius doux (8-12px)
- Shadows légères (rgba, faible opacity)
- Padding interne généreux (24-32px)

## 📱 RESPONSIVE STRATEGY

### Breakpoints Déduits
- **Mobile** : 320px - 767px (prioritaire, 70% trafic)
- **Tablet** : 768px - 1023px
- **Desktop** : 1024px+

### Adaptations Observées

**Mobile (dominant dans moodboard)**
- Stack vertical complet
- Images pleine largeur
- Padding réduit mais généreux (24-32px)
- Typographie ajustée (H1 plus petit)
- Touch-friendly (min 44x44px)

**Desktop (extrapolé)**
- Sections avec max-width container (1200px-1400px)
- Padding augmenté (80-120px vertical)
- Typographie plus grande
- Photos en grid 2 colonnes possible
- Espaces blancs encore plus généreux

## 🧩 COMPOSANTS IDENTIFIÉS

### 1. Hero Banner
**Structure** :
- Photo plein écran (background)
- Overlay subtil (rgba noir ou blanc, opacity 0.1-0.3)
- Texte centré :
  - Noms en script
  - Date en sans-serif
  - Séparateur décoratif (ligne fine ou ornement)

### 2. Countdown Timer
**Design** :
- 4 blocs : Jours | Heures | Minutes | Secondes
- Typographie tabulaire
- Labels en uppercase, letter-spacing
- Fond possible : Card avec border Sage ou transparent

### 3. Dress Code Section
**Éléments** :
- Titre "Dress Code" ou "Code Vestimentaire"
- Texte explicatif générique
- **Palette visuelle** : 5-6 cercles de couleurs alignés
- Message doux : "Si le cœur vous en dit..."
- Fond alterné (Sage ou Parchment)

### 4. Timeline / Programme
**Structure** :
- Liste verticale
- Icônes minimalistes (église, toast, danse, repas)
- Heure + Description
- Ligne connectrice verticale (Olive/Sage)

### 5. RSVP / Formulaire
**Composants** :
- Input text (nom, email)
- Radio buttons (présence oui/non)
- Textarea (message)
- Button submit (Sage background, Parchment text)

### 6. Buttons
**Primaire** :
- Background : Sage
- Text : Parchment
- Border-radius : 24px (pill shape) ou 8px (rounded)
- Padding : 12px 32px
- Hover : Olivewood background

**Secondaire** :
- Background : Transparent
- Border : 1px Sage
- Text : Sage
- Hover : Background Sage, Text Parchment

### 7. Cards
- Background : Blanc
- Border : 1px Olive (opacity 0.2)
- Border-radius : 12px
- Padding : 32px
- Shadow : 0 4px 12px rgba(0,0,0,0.06)

## 🎯 PRINCIPES DE DESIGN EXTRAITS

### 1. Minimalisme Raffiné
- Chaque élément a une raison d'être
- Pas de décorations superflues
- Focus sur l'essentiel : couple, date, informations

### 2. Hiérarchie Visuelle Claire
- Taille, couleur, spacing guident l'œil naturellement
- Progression logique : Hero → Countdown → Infos pratiques
- Contrastes maîtrisés

### 3. Nature Sophistiquée
- Palette terre/végétale
- Éléments floraux intégrés avec subtilité
- Matériaux naturels (bois, verre, tissu)
- Photos en extérieur, lumière naturelle

### 4. Intemporalité
- Pas de tendances éphémères
- Élégance classique + modernité épurée
- Design qui vieillira bien

### 5. Romantisme Discret
- Script pour noms (émotion)
- Palette douce (pas de couleurs vives)
- Photos intimistes
- Pas de clichés (cœurs, cupidons)

### 6. Accessibilité & Lisibilité
- Contrastes respectés (WCAG AA minimum)
- Typographie lisible
- Touch targets optimisés
- Mobile First
