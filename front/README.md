# Carla-Sacha-Mariage

Site vitrine pour le mariage de Carla et Sacha - 25 Juillet 2026

## Stack Technique

- React 18
- Vite
- SCSS (Sass)
- Framer Motion

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur http://localhost:5173

## Build Production

```bash
npm run build
```

Les fichiers optimisés seront générés dans le dossier `dist/`

## Déploiement Hostinger

1. Exécuter `npm run build`
2. Uploader le contenu du dossier `dist/` sur Hostinger
3. Configurer le domaine pour pointer vers index.html

## Structure

```
src/
├── components/     # Composants React
├── hooks/          # Custom hooks
├── styles/         # SCSS (variables, mixins, base)
├── assets/         # Images, fonts
├── App.jsx         # Composant principal
└── main.jsx        # Point d'entrée
```
